Краткая инструкция по удаленным репозиториям

Чтобы выгрузить с сайта github.com репозиторий, который нам нужен:

1.	Создаем рабочую папку в локале. Открываем ее в VSC.
2.	На сайте github.com в строке поиска находим нужный нам репозиторий, жмем на кнопку ‘Fork’, переходим на свой аккаунт.
3.	Копируем ссылку на репозитарий ‘Code’.
4.	В нашем терминале прописываем следующие команды:
- git clone скопированная ссылка;
- cd имя скопированного файла: - переходим в папку репозиторий (полученный с сайта)

Чтобы создать свой репозиторий на сайтe github.com:

1.	Создаем рабочую папку в локале. Инициализируем ее. Добавляем файл с данными. Сохраняем – добавляем – коммитим.
2.	На сайте жмем на кнопку ‘+’ (находится рядом с нашим профилем, вверху справа окна).
3.	В выпадающем списке выбираем пункт ‘New repository’, присваиваем имя.
4.	Далее в нашем терминале прописываем: 
‘git remote add origin ссылка на репозиторий’ – указываем куда будем выгружать; 
‘git branch -M main’ – обозначаем ветку main;
‘git push -u origin main’ – отправляем на сайт.
5.	Далее переходим на сайт github.com, обновляем страницу. 

Для отправки на сайт репозиторий с изменениями (итеративный метод работы):

1.	Добавляем данные (редактор) в репозиторий:
2.	В нашем терминале вводим команду;
git push.

            Для того чтобы выгрузить актуальную версию репозитория с сайта в локал:

1.	Вносим изменения в репозиторий на сайте, либо загружаем с другого рабочего места. Сохраняем и коммитим.
2.	В нашем терминале вводим:
git pull.

            Для работы над проектом в команде:

1.	Открываем в VSC рабочую папку.
2.	Находим на сайте нужный нам аккаунт и репозиторий.
3.	Жмем на кнопку “Fork”.
4.	Копируем адрес ссылки на репозиторий.
5.	В терминале вводим команду
git clone ссылка на скопированный репозиторий.
6.	Переходим в скопированную папку:
cd имя папки
7.	Создаем новую ветку.
8.	Переходим на созданную ветку.
9.	Создаем новый файл в репозитории. Добавляем данные в редакторе.
10.	 Сохраняем – добавляем – коммитим.
11.	 В терминале вводи команду:
git push - -set -upstream origin description
12.	 Переходим на сайт. Обновляем страницу.
13.	 Находим кнопку “Compare&pullrequest”
